# 📊 SQL-Based Instagram Engagement & Growth Analysis

## 📌 Overview

This project analyzes Instagram performance data using SQL Server (SSMS) to generate insights on engagement, reach, and follower growth.

---

## 🗂️ Dataset

The dataset consists of three tables:

* dim_dates
* fact_account
* fact_content

---

## ⚙️ Tools Used

* SQL Server (SSMS)
* SQL (Joins, CTE, Window Functions)

---

## 🔍 Key Analysis

### 1. Reach Contribution

* IG Reels contribute ~61.63% of total reach
* Most effective content format

### 2. Monthly Growth

* Peak follower growth in June (~76.94K)
* Engagement highest in mid-year

---

## 📈 Key Insights

* Reels generate highest reach
* Weekend posts perform better
* Tech categories drive engagement
* Growth peaks in May–June

---

## 💡 Business Recommendations

* Focus on Reels
* Post on weekends
* Improve low-performing content
